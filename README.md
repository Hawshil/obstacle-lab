# Parkour Movement System (UE5)

A modular, scalable parkour system built in Unreal Engine 5 focusing on fluid traversal, animation fidelity, and gameplay responsiveness.

## 🎯 Features
- Vaulting, climbing, wall-running
- Animation Blueprint integration
- Root motion + in-place hybrid system
- Speed normalization for animation sync

## 🧱 Tech Stack
- Unreal Engine 5
- C++ + Blueprint Hybrid
- Animation Blueprints
